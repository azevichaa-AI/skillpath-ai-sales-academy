# API

Основные endpoints:

- `GET /health`
- `GET /api/roles`
- `GET /api/roles/{role_id}/competencies`
- `GET /api/demo-profiles`
- `POST /api/diagnostic/run-demo`
- `POST /api/diagnostic/start`
- `POST /api/diagnostic/answer`
- `POST /api/diagnostic/finish`
- `GET /api/diagnostic/{session_id}/result`
- `POST /api/learning-track/build`
- `GET /api/learning-track/{track_id}`
- `GET /api/knowledge/search?query=...`
- `GET /api/knowledge/materials/{material_id}`
- `POST /api/control/check-answer`
- `POST /api/case/check`
- `GET /api/report/{session_id}`
- `GET /api/admin/content-status`

Swagger доступен на `/docs`.
