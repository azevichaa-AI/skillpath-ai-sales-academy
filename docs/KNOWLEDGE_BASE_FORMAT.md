# Формат базы знаний

Материалы хранятся в Markdown. Рекомендуемая структура:

```markdown
# Название компетенции

## Описание
## Практические правила
## Хороший пример
## Слабый пример
## Типовые ошибки
## Контрольный вопрос
## Эталонные элементы ответа
## Критерии оценки
```

Для подключения материала добавьте запись в `app/data/learning_materials.json` и соответствующий `.md` файл.
