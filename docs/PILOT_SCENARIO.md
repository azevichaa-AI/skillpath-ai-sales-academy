# Пилотный сценарий

1. Выбрать роль: B2B Sales Manager.
2. Настроить компетенции и стандарты компании.
3. Загрузить материалы в базу знаний.
4. Провести диагностику 10–30 сотрудников.
5. Получить карту компетенций и пробелов.
6. Назначить learning tracks.
7. Провести контроль после обучения.
8. Подготовить отчёт для руководителя и HR.
9. Принять решение о масштабировании на другие роли.
